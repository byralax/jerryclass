Three Ways to Apply CSS Styles

This page explains and demonstrates inline, internal, and external CSS styles with examples.

1. Inline CSS

This paragraph is styled with inline CSS. Inline CSS is applied directly in the HTML element using the style attribute. 

<p style="color: green; font-weight: bold;">
    This paragraph is styled with inline CSS.
</p>

2. Internal CSS

This paragraph is styled with internal CSS. Internal CSS is defined within the <style> tag inside the <head> section of the HTML document. 

<style>
    .internal-style {
        color: blue;
        font-size: 20px;
    }
</style>

<p class="internal-style">
    This paragraph is styled with internal CSS.
</p>

3. External CSS

This paragraph would be styled with external CSS if an external stylesheet were linked. External CSS is defined in a separate .css file.

The external stylesheet link would look like this in the HTML:

<link rel="stylesheet" href="styles.css">
